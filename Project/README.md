## Assignment: Task 1 – Pizza Pricing Calculator

### Project Overview
This program calculates the total cost of a pizza order using a **4-for-3 discount offer**, where the cheapest pizza is free when four pizzas are ordered.

### Objective
- Accept four pizza prices from the user  
- Apply the 4-for-3 discount correctly  
- Display the final total and the percentage discount  
- Handle invalid user input safely
